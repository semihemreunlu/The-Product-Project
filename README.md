# The-Product-Project

## Requirements
- XCode 12+
- iOS SDK 14.0+
- Swift 5+
- Cocoapods

## Architecture
- XIBs based on Autolayout
- VIPER module controller
- Service Provider pattern
- JSON codable mapping
- Unit testing
- Hero Transition

## Install
```
pod install
```
If pod specs give an error, please try
```
pod update
pod install
```
